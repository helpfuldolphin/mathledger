## Key Fields Explained

### Extraction Source
- **`"MANIFEST"`**: Panel found in `manifest.json` (preferred source)
- **`"EVIDENCE_JSON"`**: Panel found in `evidence.json` fallback (triggers advisory warning)
- **`"MISSING"`**: Panel not found (signal absent, non-error)

### Top Reason Code
- Derived from `reason_code_histogram` using canonical selection:
  1. Highest count (descending)
  2. Reason code (ascending) for tie-breaking
- Example: If `EPI_DEGRADED_EVID_IMPROVING: 2` and `EPI_DEGRADED_EVID_STABLE: 2`, selects `EPI_DEGRADED_EVID_IMPROVING` (asc order)

### Top Inconsistent Cal IDs
- Sorted ascending
- Truncated to top 3 (`top_inconsistent_cal_ids_top3`)
- Deterministic ordering for reconciliation

### GGFL Drivers
- Reason-code format only (no prose)
- Deterministic ordering: inconsistent → top reason → top cal ids
- Limited to 3 drivers maximum
