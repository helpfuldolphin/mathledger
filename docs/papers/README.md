# Papers

This directory contains academic papers describing the theory and
experimental background of MathLedger.

These papers are not required to run or verify the drop-in demo.

## Contents

| File | Description |
|------|-------------|
| `mathledger_arxiv_preprint.pdf` | arXiv preprint: "Reflexive Formal Learning: A Verifiable Learning Substrate with Ledger-Attested Feedback" |

## Note

The paper documents Phase I infrastructure validation. No capability or convergence claims are made.
